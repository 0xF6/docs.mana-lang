`Class`

The root class of most Mana class hierarchies, from which subclasses inherit a basic interface to the runtime system and the ability to behave as Mana objects.

<br/>

### Declaration


```csharp
[special, forwarded]
public class Object
{
    ...
}
```


### Methods


```csharp
[virtual]
public toString(): String;
```
